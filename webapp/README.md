## 📝 早退記錄使用說明

### 教師提交申請
1. 點擊「早退記錄」標籤
2. 點擊「新增早退申請」
3. 填寫信息：
   - 選擇教師姓名
   - 選擇所屬學部
   - 選擇早退日期
   - 選擇離開時間
   - 選擇早退原因
   - 填寫詳細說明（選填）
4. 點擊「提交申請」

### 主任審批
1. 在早退記錄列表中找到待審批的記錄
2. 點擊審批圖標（📋）
3. 查看詳情
4. 選擇「批准」或「拒絕」
5. 可填寫審批意見（拒絕時必填）

### 早退原因選項
- 事務堂
- 外出比賽
- 教務會議
- 家校會議
- 教師培訓
- 身體不適
- 家庭事務
- 其他

### 查詢和統計
- 按狀態篩選（待審批/已批准/已拒絕）
- 按教師篩選
- 按日期範圍查詢
- 統計卡片顯示各狀態數量

## 💬 常見問題

**Q: 如何切換離線/網頁版本？**
A: 離線版本直接打開 index.html；網頁版本需啟動服務器。

**Q: 離線版本的數據會丟失嗎？**
A: 數據保存在瀏覽器中，清除瀏覽器數據會影響。建議定期導出CSV備份。

**Q: 多人使用時數據會同步嗎？**
A: 網頁版本使用 WebSocket 實時同步數據。

**Q: 早退記錄可以刪除嗎？**
A: 可以。在記錄列表中點擊刪除圖標即可。

**Q: 審批後還能修改嗎？**
A: 審批後無法修改狀態，但可以刪除記錄。
